###################
OpenEduCases
###################

A ferramenta web de apoio à Aprendizagem baseada em Casos, denominada OpenEduCases permite que educadores realizem a inserção, o uso e a busca de casos, bem como a visualização de estratégias para usar tais casos em sala de aula.

Desenvolvida em período de estágio no IFSULDEMINAS - Campus Muzambinho
